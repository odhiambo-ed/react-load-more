# React Load More

## Description

> This project is a simple implementation of a "Load More" button component in React. It demonstrates how to manage state and handle user interactions to dynamically load more content.

## Live Demo Link

[Project Link](https://github.com/odhiambo-ed/react-load-more)

## Features

- Dynamically loads more content when the "Load More" button is clicked.
- Manages state efficiently to keep track of loaded content.
- Simple and reusable component structure.

## Get Started

### Run

```
To get a clone of the project, run `https://github.com/odhiambo-ed/react-load-more.git`
```

```
cd /react-load-more
```

```
npm install
```

```
npm run dev
```
